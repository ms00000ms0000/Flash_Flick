This Android application, built in **Kotlin using Android Studio**, is a **Flashcards Quiz App** designed for both fun and learning. Users can **add, edit, or delete questions** to create personalized quiz sets based on their interests and study goals. The app provides an engaging way to **test knowledge, revise concepts, and enhance memory retention** through interactive flashcards. Whether used for educational purposes or casual quizzes, it offers full flexibility to customize content anytime. Simple, lightweight, and user-friendly, this project showcases efficient data handling, smooth UI design, and interactive quiz functionality for an enjoyable learning experience.
